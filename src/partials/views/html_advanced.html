Advanced HTML

This page will cover some of the more advanced aspects of HTML

The <div> tag

The <div> tag means division it is used for separating out elements on a page. That means that different styles can be applied to different divs. Applying styles to web pages will be discussed in later sections.

Tables

HTML can be used to display tables on the page. Tables are comprised of several different tags. To create a table, you start with a <table> tag. Tables are made up of both rows(going across) and columns(going down).

The rows are set up using the <tr> tag. The number of columns are set up using either <th> or <td> tags. <th> tags are used for the table headers while <td> tags are used for the data. Below is a simple table made using this method.
