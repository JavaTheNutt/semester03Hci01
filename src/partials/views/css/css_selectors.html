CSS Selectors

This section will explain what classes and id's are used for when using CSS.

Classes Id's

Classes

Classes are added to HTML elements to ensure that when CSS styles are applied, they are only applied to the elements that also have the class applied. Consider the below HTML: 

{{classes.pre01}}

Now we can select the <p> tag with the class added without selecting any p tags without the class. This is done like this: 

{{classes.pre02}}

The above rules will set the color of any <p> tag to blue. Then it will specify that any <p> tag with the para class attached (notice p.para in the stylesheet) will be red. The button below will show you what happens when you combine the html and css.

Id's

An id is similar to a class except that it should only be added to a single element on a page. It is more commonly used in JavaScript where you would need to access a specific element on a page. To use an id in HTML, add the following code: 

{{ids.pre01}}

Notice how we used id="para" instead of class="para". Below is an example of how to implement this code 

{{ids.pre02}}
