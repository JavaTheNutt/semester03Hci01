Including CSS

There are several ways to include CSS in your website. The most common way is by using an external stylesheet, but we will explain them all here.

Show Internal Show External Show Inline

Inline Styling

Inline styling is the easiest way to add style to a page, but it is very limited in what it can do. Inline styling is applied directly to the element. Inline styling can be useful when a small amount of style is needed. An example would be, if you wished to turn a paragraph red(remember our <p> tag!), you would write the tag as follows: 

<p style="color:red">Some text here</p>
