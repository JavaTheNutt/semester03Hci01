CSS

CSS stands for Cascading Style Sheets. CSS is used to apply styles to websites.

When HTML is used to create a page, the data is presented without any formatting or style applied to it. This is where CSS comes in.

CSS can be used to apply many different types of styles to websites such as colors, borders and other styles. The most recent version is CSS3, which works well with HTML5.

It is important to remember that CSS can't be used to add any data to the screen. CSS is only used to apply formatting to any data already placed on the page.
