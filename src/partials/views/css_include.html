Including CSS

There are several ways to include CSS in your website. The most common way is by using an external stylesheet, but we will explain them all here.

Inline Styling

Inline styling is the easiest way to add style to a page, but it is very limited in what it can do. Inline styling is applied directly to the element. Inline styling can be useful when a small amount of style is needed. An example would be, if you wished to turn a paragrapgh red(remember our <p> tag!), you would write the tag as follows: 

<p style="color:red">Some text here</p>

Internal Stylesheet

Another easy way to add style to a page is to put an internal stylesheet inside the head of the page. This is a neater method than inline styling. To use an internal stylesheet in the head section of your page. Below is an example of an internal stylesheet. 

<style>
    p: {
        color:red;
        font-size:12px;
    }
    h1: {
        color:blue;
    }
</style>

The above code would be written in the head tag and would set the font color of all paragraphs to red and the font size to 12 pixels. This will also set the font color of any <h1> tag to blue.
