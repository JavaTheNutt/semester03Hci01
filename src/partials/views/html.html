About HTML

HTML is the backbone of the Web. HTML stands for Hyper Text Markup Language. It is used to display information on a web page. It is used for placement of data on a page. The current version of HTML is HTML5. This is what will be shown here.

HTML is written between what are called tags. A set of tags looks like this: <tag name here>. Examples of tags would be <p>, <br> and <h1>

Most tags come in pairs. Whenever a tag that works as part of pair is opened, the equivalent closing tag must be inserted. To close a tag you simply need to put the tag name with a forward slash(/) before it, like this: <h1>Some text here</h1>.

Some tags do not need to be closed. We call these empty tags. Examples of empty tags are: <br>, <hr>, <img> and <input>.

Another example of an empty tag is <!DOCTYPE html>. This is a very special tag that is placed at the very top of each HTML page. This specifies to the browser that the HTML used is HTML5.
