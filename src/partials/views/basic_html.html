Some basic html tags

This page will explain some of the basic html tags and how they should be used.

The <p> tag

This is the most basic tag. It is a paragraph tag. This tag is used for placing basic paragraphs of text on the screen.

The heading tags

There are six types of heading in html. They range from <h1> to <h6>, with <h1> being the largest and <h6> being the smallest.

The <hr> tag

The <hr> tag stands for Horizontal Rule and is used for placing a horizontal line across the page.
