HTML Layout

This page will show the building blocks of HTML that every page must have.

The <!DOCTYPE html> tag

This tag is placed at the start of the HTML page. It tells the browser that you are using HTML5. This is what is known as an empty tag, which means that it doesn't have to be closed.

The <html> tag

This pair of tags goes around the entire page, everything you write after this will go inside these tags. Remember, you can close a tag like this </html>.

The <head> tag

This tag is used to hold some of the data about the page. This tag goes inside of the <html> tag. An example of this is the <title> tag. This tag is used to set the title of the page. The title is the name that you see in the top of the tab.

The <body> tag

This tag is where the actual content of the web page goes. This tag also goes inside of the <html> tag. It goes below the <head> tag.
