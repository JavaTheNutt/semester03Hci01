Hyperlinks

The <a> tag

The <a> tag is used for hyperlinks. A hyperlink is a link that links you to something else on the internet. An example would be how I can make this word link to a picture of a cat. Another example is the all of the navigation buttons on this site. They open up pages because they are <a> tags! 

{{examples.aTag}}

The above code is an example of an <a> tag. The href attribute is the hyperlink reference and is the URL of where you want to go. (A URL is an address on the web). The target="_blank" makes sure that this link is opened in a new window. If you are looking to link to something within your site, you just need to put the path to that file in the href attribute.
