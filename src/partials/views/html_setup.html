Setup for Webdesign

In order to begin designing webpages with HTML, no special tools are required. All you need is a text editor(such as Notepad) and a browser (such as Google Chrome). It may be difficult to write your HTML in Notepad but there are lots of free alternatives, such as Notepad++ and Atom. A good paid alternative is Sublime Text, which has an unlimited free trial. For a truly professional tool you could try Dreamweaver or WebStorm, but these programs are quite expensive and are really not necessary for beginner designers.

For basic HTML design, any browser will work. Internet Explorer already comes pre-loaded on every Windows machine. Safari is the default browser for OSX. Most Linux distros come with FireFox pre-installed. Any of these defaults will work fine, although Internet Explorer is not fully compatible with all features of HTML5. Good alternatives would be Google Chrome or Mozilla FireFox.

Once you have your text editor and browser installed, you're good to go!
