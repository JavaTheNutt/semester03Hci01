Setup for HTML

In order to begin designing webpages with HTML, no special tools are required. All you need is a text editor(such as Notepad) and a browser (such as Google Chrome).

It may be difficult to write your HTML in Notepad but there are lots of free alternatives, such as Notepad++ , Atom or sublime .

For basic HTML design, any browser will work. I would recommend either Google Chrome or Mozilla FireFox.

Once you have your text editor and browser installed, you're good to go!

To test that everything is working correctly, copy the following text into your text editor and save it as index.html (Ensure that the dropdown menu is set to "all files" or html document). 

<!DOCTYPE html>
<html>
    <body>
        <p>Hello, World<p>
    </body>
</html>

Once you have the file saved, open it with a browser. If you typed it correctly, you should see the words "Hello, World!" printed at the top of the screen.

Now you've designed your first website! Congratulations!!
